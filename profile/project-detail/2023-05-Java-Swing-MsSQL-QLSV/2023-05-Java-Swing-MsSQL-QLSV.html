PROJECT-QLSV-JAVASWING-MSSQL

Sử dụng Java Swing và Microsoft SQL Server để tạo ra một giao diện quản lý sinh viên.

Tiến Hành Cài Đặt

Về Phía SQL

  1. Sử dụng "SQL Server Management Studio Management Studio 19" và cài đặt cấu hình (login sa và password 12345).
  2. Chạy file QUANLYSINHVIEN.sql để tạo database.
  3. Sử dụng "SQL Server 2022 Configuration Manager" để đổi cổng TCP/IP thành tất cả Port 1433.

Về Phía Java

  1. Ở IntelliJ IDEA vào mục database bên phải màn hình.
  2. Thêm các nguồn của MSSQL như sau:

Nó trả về 1 đường link Url ta sẽ tận dụng đường link đó để kết nối code với MSSQL.

  1. Chạy file Login và nhập Tài Khoản "admin", Mật Khẩu "12345".
