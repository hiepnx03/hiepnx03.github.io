Cài Đặt Chương Trình
  1. Phần mềm sử dụng Visual Studio 2019 và SQL Server Management Studio 19.
  2. Cấu hình tài khoản SQL Server là 'sa' và mật khẩu là '12345'.
  3. Tải các thư viện có hỗ trợ Winform ở Visual Studio 2019 (tải toàn bộ cho chắc).
  4. Tạo 2 project: 1 là Admin với Project1.
  5. Copy các file .cpp, .h, .resx và dán vào các project.
